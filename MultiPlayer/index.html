Explanation

This Java applet sends playersquery and a question mark to the Perl script asking it how many hypothetical players are waiting. At which point the Perl script will decode the query and send back a new line '\n' and then a 1 back to the Java script. 1 being how many hypothetical players are waiting. The Java applet space is just an analysis point after the Perl script has been queried and attempts to decode the information.

If you wish you can see other messages while initializing and sending data to the Perl script by activating the java console in your web browser.
